### 1. Two-Screen Reporting Flow

- Single decision per screen

**Screen 2 ‚Äì Confirm & Submit**

---

### 2. Accessibility-First Design (WCAG 2.1 AA+)

Built-in support for:
- Keyboard-only navigation
- Screen readers (ARIA roles, live regions)
- High contrast mode
- Reduced motion (respects OS preferences)
- Adjustable text sizes
- Large touch targets (44px+)

Accessibility is treated as a **product requirement**, not a feature.
